# 370 Canvas Dupe

## Description

Our project is a basic example of the groundwork of a LMS system, like Instructure Canvas.

## Installation & Running

**Step 1:** Clone the repository [https://github.com/zacheri04/370_CanvasDupe](https://github.com/zacheri04/370_CanvasDupe)

**Step 2:** Ensure you have the php command line utility installed, check with

```
php -v
```

should yield version number 8.4.5 or similar.

**Step 3:** Start the php server in the public directory

```
cd public && php -S localhost:8000
```
